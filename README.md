# TaskFlow SaaS

Demo de portfólio para gestão de tarefas com visual SaaS moderno.

## Features

- dashboard operacional
- múltiplos responsáveis por tarefa
- tags, checklist e notas
- comentários e atividade recente
- drag and drop entre colunas
- login de demonstração
- persistência local com `localStorage`

## Stack

- Next.js
- React
- TypeScript
- Tailwind CSS

## Publicação

O projeto está preparado para export estático e publicação no GitHub Pages.

Variáveis opcionais em `.env.example`:

- `NEXT_PUBLIC_BASE_PATH`
- `NEXT_PUBLIC_GITHUB_PAGES_URL`
